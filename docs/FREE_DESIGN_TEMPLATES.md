# Free, Open-Source Design Templates (Verified)

CPO requested research on free, top-tier design templates for landing page repositioning. All below are verified free and open-source (MIT or equivalent). No paid options included.

---

## Full Landing Page Templates (Next.js / React)

### Launch UI
- **Link:** https://github.com/launch-ui/launch-ui  
- **Demo:** https://launchuicomponents.com/preview  
- **Stack:** Next.js 15, Shadcn/ui, Tailwind, TypeScript  
- **License:** MIT  
- **Best for:** Developer tools, AI apps, SaaS — matches infra positioning  
- **Notes:** Navbar, Hero, FAQ, Stats, CTA, Footer. Pro version has more; base is fully free.

### Shadcn Landing Page (nobruf)
- **Link:** https://github.com/nobruf/shadcn-landing-page  
- **Stack:** Next.js, Shadcn, TypeScript, Tailwind  
- **License:** MIT  
- **Stars:** 1.1k+  
- **Sections:** Navbar, Hero, Sponsors, Benefits, Features, Testimonials, Team, Pricing, FAQ, Footer

### Shadcn Landing Page (leoMirandaa)
- **Link:** https://github.com/leoMirandaa/shadcn-landing-page  
- **Demo:** https://shadcn-landing-page.vercel.app  
- **Stack:** React, Vite, Shadcn, TypeScript, Tailwind  
- **Stars:** 1.8k+  
- **Sections:** 16 sections (Navbar, Hero, Sponsors, About, Stats, Features, Services, Testimonials, Team, Pricing, FAQ, CTA, Footer, etc.)

## Component Libraries (Copy-Paste)

### HyperUI
- **Link:** https://hyperui.dev  
- **GitHub:** https://github.com/markmead/hyperui  
- **License:** MIT  
- **Stars:** 11k+  
- **Components:** 226+ (Hero, CTA, Marketing, Application, Neobrutalism)  
- **Notes:** Copy-paste HTML/Tailwind; view as JSX. Completely free, no attribution required. Works with Tailwind v4.

### Tailkits Free
- **Link:** https://tailkits.com/ui/free/  
- **Components:** 30+ (Hero, Features, Stats, FAQ, Pricing, Navbar, Footer, etc.)  
- **Notes:** HTML + Tailwind; MIT. Free subset only; paid tier exists for more, but free set is fully usable.

---

## Quick Picks for Reflect Memory

- **Infra / dev-tools vibe:** Launch UI — built for “Developer Tools, AI-Powered Applications”  
- **Minimal, clean:** nobruf or leoMirandaa shadcn templates  
- **Maximum flexibility:** HyperUI — mix and match 226 components
